## SET\_CHANNEL\_MUTED

Sets Mute for a Channel.


### Name
`SET_CHANNEL_MUTED(roomId: Long, channelId: Int, muted: Boolean)`


| Parameter | Type | Description              |
| --------- | ---- | ------------------------ |
| roomId    | Long | ID value of the room.    |
| channeld  | INT  | ID value of the channel. |
| muted     | Bool | Mute state (t/f).        |
